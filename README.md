# Registration Assistant

This prototype assists people to find out which documents they need for their registration at a BÃ¼rgeramt in Berlin.

## Installation

Make sure to use Node.js version 20.6.1 or higher.

```bash
npm install
```

## Development

```bash
npm run dev
```

## Tests

```bash
npm run test
```
